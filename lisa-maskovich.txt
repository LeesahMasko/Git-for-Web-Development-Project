1. What is Semantic HTML? 
	Elements that bring meaning, like; <form>, <table>, <article>
    2. What is HTML used for? 
	It is used as the standard markup language for information intended to be displayed in a web browser.
    3. What is an attribute and where do we put it? 
	Attributes provide additional information about the HTML element.  They are placed after the element.  
	<img src="img_girl.jpg"> In this case the src is the attribute

	The H1 tag is a header tag used especially for the most important title.  Possibly the first header element.  It the most important information used by search engines.

    7. What is an id?
	An ID attribute is used to specify a unique ID for an HTML element, making it easier to call out in the style CSS 
    8. What elements can I add an id to? 
	It can be used on any HTML element.
    9. How many times can I use the same id on a page? 
	An HTML element can only have one unique ID that belongs to that singular element.
    10. What is a class? 
	The HTML class is for elements that are the same and will have the same styles implemented. 
    11. What elements can I add a class to?
	The class attribute can be used on any HTML element.
